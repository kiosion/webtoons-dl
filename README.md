# webtoons-dl
Simple script to pull &amp; compile comics from Webtoons. Downloads all episodes of a comic, and compiles panels into single JPG images.

## Usage
`python3 webtoons-dl.py [comic url] [--dir [directory]] [--zip]`

## Dependencies
* Python 3
* Requests
* Pillow
* BeautifulSoup4
